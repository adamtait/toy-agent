# Component-Level Documentation

This directory contains detailed documentation for each of the major components of the autonomous agent.

## Components

- [**Agent Core (`agent.md`)**](./agent.md): Describes the main `ReactAgent` class, the reasoning-action loop, and the overall orchestration of the agent.
- [**LLM Client (`llm.md`)**](./llm.md): Explains how the agent interacts with Large Language Models, including prompt construction and response parsing.
- [**Tools (`tools.md`)**](./tools.md): Provides a detailed reference for all available tools that the agent can use to interact with the code repository.
- [**Main Entry Point (`main.md`)**](./main.md): Covers the command-line interface, configuration, and the agent's initialization process.
